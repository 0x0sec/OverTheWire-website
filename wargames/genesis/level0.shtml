Level 0 - Logging in and out

The most basic of skills is logging in to a wargame using SSH. Except for some special levels in certain wargames, logging in with SSH is a requirement to get to the next level. So let's start here.

If you don't already have an SSH-client installed, install one first. On most Linux installations an ssh-client is installed by default. In Windows, we can recommend PuTTY. Installation and usage of this software is explained elsewhere. We will assume you use Linux.

Loggin in

To login to the first level, login with username "level0" to server "genesis.overthewire.org" like this: 

ssh level0@genesis.overthewire.org

You'll be prompted for a password, that password is "YFGHn9EgB1rOV" without the quotes. After logging in, your screen will look similar to this: 

deepstar@phoenix:~$ ssh level0@genesis.overthewire.org
level0@genesis.overthewire.org's password: 

Last login: Sat Jan 31 11:36:34 2009 from 192.168.1.1

The password for level1 is LW36aSLsfW2D9
level0@genesis:~$

Notice that my host is called "phoenix" and my username is "deepstar". After entering the correct password for level0, information about the last login to this account is shown. Below that, the password for level1 is printed. This will only be the case for this level, other levels will require you to do other things to obtain the next level's password.

Logging out

To logout of this level, type "exit" 

level0@genesis:~$ exit
logout
Connection to genesis.overthewire.org closed.
deepstar@phoenix:~$

Try it!

Now it's your turn. Log in to the level0 account and obtain the level1 password (which is different from the one mentioned above). You can use that password to log in to level1. See you there!
