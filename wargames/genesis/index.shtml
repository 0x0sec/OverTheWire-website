Genesis

Learning the basics

If you're new to wargames, this is the place to start. While other wargames on OverTheWire focus on security concepts, this wargame will focus on the tools used to play the other wargames.

In other words, the basics.

Structure

To the more experienced player this "game" will look terribly simple, but that is the idea anyway. Like most wargames this one also consists of a chain of linked levels, having to solve a level before getting access to the next. Because this game tries to teach the basics, each level will consist of an example and its solution, aswell as an excersise. Solving the excersise will give access to the next level.

Not enough ?

This game is probably incomplete. We can't possible provide you with all the basics needed to play the other games. However, if you feel that a certain skill is needed that we have missed somehow, let us know.
